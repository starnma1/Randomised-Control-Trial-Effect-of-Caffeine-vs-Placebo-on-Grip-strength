# Randomised-Control-Trial-Effect-of-Caffeine-vs-Placebo-on-Grip-strength

This project analyses the effect of acute caffeine ingestion on maximal voluntary grip strength in a double-blind randomised controlled trial (n=13). Participants were assigned to either caffeine or vitamin C (placebo) and grip strength was measured before and after intervention. A linear mixed model was fit using glmmTMB, with pre-intervention grip strength and weight as covariates and participant as a random intercept. Secondary analyses examined whether habitual caffeine intake moderated the effect, and whether the intervention had any detectable effect on systolic blood pressure and heart rate.

Data is private.
